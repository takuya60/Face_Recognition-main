# 本地人脸识别门禁系统

本项目使用正点原子阿尔法i.MX6ull开发板开发，使用fsl-imx-x11 SDK

基于Linux内核，Qt界面开发，openCV进行人脸识别，SQLite进行本地数据库写入

使用qmake编译


## 功能
- 自动识别人脸并比对
- LCD图形界面
- 识别日志本地储存，掉电不丢失，可导出读取
- 串口控制设备开门
- 添加新人脸

*所有过程均本地运行，不需要联网*


## 项目框架

| 文件(夹) | 内容 |
| -------- | ------ |
| bin | 可执行文件 |
| obj | 编译产生中间文件(忽略) |
| inc | 头文件 |
| src | 源文件 |
| face.pro | 配置文件 |
| sour.sh | 环境配置脚本 |
