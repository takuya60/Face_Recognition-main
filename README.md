# 人脸识别后端 API 集成

这是人脸识别模块的后端（Backend）代码。这个后端被设计为在 Qt 的一个独立工作线程中运行，以确保 UI 完全不会卡顿

基于Linux内核，Qt界面开发，openCV进行人脸识别

使用qmake编译


## 功能
- 自动识别人脸并比对
- LCD图形界面
- 识别日志本地储存，掉电不丢失，可导出读取
- 串口控制设备开门
- 添加新人脸

*所有过程均本地运行，不需要联网*


## 项目框架

| 文件(夹) | 内容 |
| -------- | ------ |
| bin | 可执行文件 |
| obj | 编译产生中间文件(忽略) |
| inc | 头文件 |
| src | 源文件 |
| face.pro | 配置文件 |
| sour.sh | 环境配置脚本 |
